git init (inicia el programa)
git add nombre.txt (agrega a stage el archivo)
git commit -m "texto" (hace commit del archivo)
git diff (muestra los cambios que tienen los archivos)
git diff --staged (muestra los cambios que hay en los archivos en staged)
git commit (muestra los commits que hay en un file en VSC)
git log --oneline (muestra los logs con su respectivo "hash" donde estan los commits hechos)
git branch (muestra las ramas)
git checkout -b nombreDeLaNuevaRama (crea una rama nueva)
git checkout nombreDeLaRama (sirve para trasladarse entre ramas)
git push -u origin nombreDeLaRama (envía los archivos que tiene esta rama a la nube)
git merge nombreDeLaOTRARama (hace merge de los cambios que tiene una rama menor o mayor)
git checkout -- . (trae los archivos que tenía en el último commit)
(crear rama para uno trabajar en el proyecto)
git config –list (muestra la información del archivo de configuración)
git remote set-url origin url.del.repositorio.git.com (setea o cambia el repositorio que se usa)
git pull origin main (jalar cambios del branch main) 
git status (muestra el arbol de los files added)
git push origin main
